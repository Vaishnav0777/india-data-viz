\# Data Dictionary



\## india\_states.csv



| Column | Type | Description | Example |

|--------|------|-------------|---------|

| state | string | Name of the Indian state | Maharashtra |

| population\_millions | float | Population in millions | 112.4 |

| area\_sq\_km | integer | Geographic area in sq km | 307713 |

| density\_per\_sq\_km | integer | People per sq km | 365 |

| literacy\_rate | float | Literacy percentage | 82.3 |

| sex\_ratio | integer | Females per 1000 males | 929 |

| gdp\_billion\_usd | float | State GDP in billion USD | 400.0 |

| capital | string | State capital city | Mumbai |

| region | string | Geographic region | West |



\## Regions



\- North: UP, Rajasthan, Punjab, Haryana, HP, Uttarakhand

\- South: Tamil Nadu, Karnataka, AP, Telangana, Kerala

\- East: Bihar, West Bengal, Odisha, Jharkhand

\- West: Maharashtra, Gujarat, Goa

\- Central: MP, Chhattisgarh

\- Northeast: Assam, Tripura, Meghalaya, Manipur, Nagaland, Mizoram, Arunachal Pradesh, Sikkim

## Derived fields

- `gdp_per_capita` — computed at analysis time as `gdp_billion_usd * 1000 / population_millions`, expressed in USD per person. Not stored in the CSV; generated by `src/plot_correlation.py`.

## Notes on correlations

Population, area, and GDP move together strongly (population–GDP ≈ 0.77), reflecting that larger, more populous states tend to have larger economies. Literacy correlates more with GDP per capita than with absolute GDP, pointing to development quality rather than raw size.
